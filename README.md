## 프로젝트를 진행하면서 느낀점
Z-Index를 활용한 배치가 어려웠다. 사용할 때는 자연스러운 기능이었지만, 직접 구현할 때 특정 화면에서만 보이는 UI를 그리고 배치하는 것도 신선했던 것 같다. 
그리고 영역 잡는게 까다로웠다. 아직 정렬, 배치가 익숙하지 않다고 느꼈다.
